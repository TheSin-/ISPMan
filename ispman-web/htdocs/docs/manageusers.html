Manage users
Follow these steps to add a new user
To manage users of a domain, you must choose a domain from domain selection box, wait till it refreshes and then use the buttons as outlined.
  • Surname
  • Firstname
  • Password
  • Primary Mail Address
  • Primary Mail Box Address
  • Mail Forwardings
  • Mail Aliases
  • Reset - Delete User - Update User
    • As a domain increases in size, you may need to search for a user by First Name, Last Name or User ID.
