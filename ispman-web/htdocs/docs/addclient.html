Adding a Client
To add an client, you must be logged in as a reseller or be an ISPMan administrator with an authority level of at least 20.
Each client is associated with a reseller, who can also manage the client's domains. This allows the reseller to provide support to the client if a question arises. If there are no resellers in the system, it is impossible to add a client.
To add a client:
  • Click on Clients, then click on "Add a client"

  • Select the reseller for this client.
  • Enter the username, password and other information about the client.
  • Set a maximum number of domains for the client.
  • Click "Add this client" when finished
