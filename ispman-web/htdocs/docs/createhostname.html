Create a DNS hostname
You can make a DNS hostname by adding a DNS "A Record".
Here are the steps:
  • Select the domain to manage
  • Under Manage DNS, click A Records
  • You'll see a list of all Address Records for the domain. To add a new record, type in the hostname in the first field and the IP address in the second field.
  • Check the Rev. checkbox if you want a Reverse DNS record to be created.
  • Click Save.
  • To apply your change, click Commit session.
NOTE: You will only see the Manage DNS section on a domain that is a primary ISPMan domain. Manage DNS will not appear on replica domain, secondary/slave domains, or nodns domains.
