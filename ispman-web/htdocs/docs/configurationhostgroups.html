Configuring Hostgroups
ISPMan uses hostgroups to know what hosts to update when a change occurs. To make ISPMan fully functional, there must be at least 1 host in each hostgroup.
To add Hosts to a group
  • Edit the group to change
  • The screen will update, showing details about that group.
  • Highlight the first host to be a group member
  • Select any additional hosts, by holding the Ctrl (Control) key down while clicking
    • NOTE: On a Macintosh, hold down the Command key to select multiple hosts.
  • Click Update when finished
