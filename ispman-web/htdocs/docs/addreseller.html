Adding a Reseller
To add a reseller, you must be an ISPMan administrator.
Each reseller can create clients and fully manage their clients domains.
To add a reseller:
  • Click on Resellers, then click on "Add a reseller"

  • Enter the username, password and other information about the reseller.
  • Set a maximum number of clients for the reseller.
  • Click "Add this reseller" when finished
