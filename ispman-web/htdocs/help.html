ISPMan Online Help
ISPMan is a powerful interface for controlling email, users, DNS, and webhosting.
Most tasks start by selecting the domain. This opens details about the domain, such as users and hosting information. Select the task to accomplish below. When finished requesting changes, click "Commit Session". "Commit Session" tells ISPMan to schedule your changes now. Changes should be complete within 1-5 minutes.
Here are the most common tasks done with ISPMan:
  • Reset FTP Password to upload a website
  • Add an email account
  • Reset a user's email password
  • Forward email for a user
  • Make a distribution email list
  • Create a DNS hostname for an IP address
  • Add a website/virtualhost
Here is information about each section of ISPMan:
  • Add a domain
  • Edit a domain
  • Manage DNS
  • Manage users
  • Manage mailgroups
  • Manage virtualhosts
